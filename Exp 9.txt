Q=
1. Read an image file into MATLAB. 2. Display the image using imshow. 3. Convert RGB image to grayscale using rgb2gray. 4. Convert RGB to HSV using rgb2hsv. 5. Display results. 6. Resize image to half size. 7. Crop a region of interest. 8. Display results. 9. Convert image to grayscale. 10. Normalize using mat2gray. 11. Display results.

A=
clc; clear; close all;

% 1. Read an image file
img = imread('peppers.png');   % You can replace 'peppers.png' with your image file

% 2. Display the original image
figure('Name','Original Image');
imshow(img);
title('Original RGB Image');

% 3. Convert RGB image to Grayscale
gray_img = rgb2gray(img);

% 4. Convert RGB to HSV
hsv_img = rgb2hsv(img);

% 5. Display results (RGB, Grayscale, HSV)
figure('Name','Color Conversions');
subplot(1,3,1); imshow(img); title('RGB Image');
subplot(1,3,2); imshow(gray_img); title('Grayscale Image');
subplot(1,3,3); imshow(hsv_img); title('HSV Image');

% 6. Resize image to half size
resized_img = imresize(img, 0.5);

% 7. Crop a region of interest (ROI)
% For example, crop a region starting from (50,50) with width=100 and height=100
cropped_img = imcrop(img, [50 50 100 100]);

% 8. Display resized and cropped results
figure('Name','Resized and Cropped');
subplot(1,2,1); imshow(resized_img); title('Resized Image (Half Size)');
subplot(1,2,2); imshow(cropped_img); title('Cropped Region');

% 9. Convert image to Grayscale again (if needed)
gray_again = rgb2gray(img);

% 10. Normalize grayscale image using mat2gray
normalized_img = mat2gray(gray_again);

% 11. Display normalized result
figure('Name','Normalization');
subplot(1,2,1); imshow(gray_again); title('Grayscale Image');
subplot(1,2,2); imshow(normalized_img); title('Normalized Image (mat2gray)');


EXERCISE 2:
1. Load your own image and display it in grayscale and HSV color spaces. 2. Resize an image to double its size and compare with the original. 3. Crop an image to extract only the face (or any object of interest). 4. Normalize an image and verify intensity values using min() and max().

A=
clc; clear; close all;

%% 1. Load your own image
% Replace 'myimage.jpg' with the actual image filename
img = imread('bird.jpg');

% Display original image
figure('Name','Original Image');
imshow(img);
title('Original RGB Image');

% Convert to Grayscale
gray_img = rgb2gray(img);

% Convert to HSV
hsv_img = rgb2hsv(img);

% Display grayscale and HSV versions
figure('Name','Grayscale and HSV');
subplot(1,2,1);
imshow(gray_img);
title('Grayscale Image');

subplot(1,2,2);
imshow(hsv_img);
title('HSV Image');

%% 2. Resize image to double its size
resized_img = imresize(img, 10);  % Resize by a scale factor of 2

% Display and compare original vs resized
figure('Name','Resizing Comparison');
subplot(1,2,1);
imshow(img);
title('Original Size');

subplot(1,2,2);
imshow(resized_img);
title('Resized Image (Double Size)');

%% 3. Crop a region of interest (face or any object)
% You can select interactively by dragging a rectangle over the image
figure('Name','Crop Region');
imshow(img);
title('Select Region to Crop');
cropped_img = imcrop(img,[100 100 100 100]);  % Interactive cropping

% Display cropped region
figure('Name','Cropped Image');
imshow(cropped_img);
title('Cropped Region (Object of Interest)');

%% 4. Normalize an image and verify intensity values
% Convert to grayscale first if not already
gray_img = rgb2gray(img);

% Normalize grayscale image (intensity between 0 and 1)
normalized_img = mat2gray(gray_img);

% Display normalized image
figure('Name','Normalized Image');
imshow(normalized_img);
title('Normalized Grayscale Image (0â€“1 Range)');

% Verify intensity values
min_val = min(normalized_img(:));
max_val = max(normalized_img(:));

% Display results in Command Window
fprintf('Minimum intensity value after normalization: %.4f\n', min_val);
fprintf('Maximum intensity value after normalization: %.4f\n', max_val);
